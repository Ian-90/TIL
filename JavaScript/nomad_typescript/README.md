## Setting
* 설치
```
yarn global add typescript
```

* [tsconfig.json](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) 추가 - TS가 어떻게 JS로 변환할지 알려주는 것

* 컴파일 방법
  * global 설치
  ```
  tsc [파일명]
  ```

  * local 설치
  ```
  npx tsc [파일명]
  ```
